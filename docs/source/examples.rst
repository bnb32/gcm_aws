Run Simulations
===============

Running a CESM simulation for 300Ma BP
**************************************

.. code-block:: bash
    python scripts/run_cesm.py -year 300.0 -run_all


Running an Isca simulation for 300Ma BP
***************************************

.. code-block:: bash
    python scripts/run_isca.py -year 300.0
