Run CESM simulation

Run CESM

usage: python scripts/run_cesm.py [-h]
                                  [-exp {cam_clmCN_docnSOM,cam_clmCN_docnDOM,aqua}]
                                  [-multiplier MULTIPLIER]
                                  [-co2_value CO2_VALUE]
                                  [-sea_level SEA_LEVEL]
                                  [-max_depth MAX_DEPTH] [-year YEAR]
                                  [-case CASE] [-ntasks NTASKS]
                                  [-nthrds NTHRDS] [-start_date START_DATE]
                                  [-step_type STEP_TYPE] [-nsteps NSTEPS]
                                  [-restart] [-setup] [-build] [-run]
                                  [-run_all] [-remap] [-remap_hires] [-timing]
                                  -config CONFIG

Named Arguments

-exp

Possible choices: cam_clmCN_docnSOM, cam_clmCN_docnDOM, aqua

Default: “cam_clmCN_docnDOM”

-multiplier

CO2 Multiplier

Default: 1.0

-co2_value

CO2 Value

-sea_level

Sea level to use for simulation. Relative to current era. So to lower the level use a negative number.

Default: 0

-max_depth

Max ocean depth. This is the number of meters below sea level at which the ocean depth will be capped for the simulation.

Default: 1000

-year

Years prior to current era in units of Ma.

Default: 0

-case

Default: “test”

-ntasks

Default: 96

-nthrds

Default: 8

-start_date

Default: “0001-01-01”

-step_type

Default: “ndays”

-nsteps

Number of steps to run simulation. If step_type is days then this will be the number of days to run

Default: 300

-restart

Whether to restart simulation from previous run

Default: False

-setup

Whether to run experiment setup. Needs to be run at least once before the simulation

Default: False

-build

Whether to build the experiment directory. Needs to be run at least once before the simulation

Default: False

-run

Run the simulation.

Default: False

-run_all

Default: False

-remap

Default: False

-remap_hires

Default: False

-timing

Default: False

-config

Path to configuration file. Needs to have the parameters defined in environment/config.py. Needs to be in either json format or a .py file.
